# Hacking-Labs

Repositorio que documenta el proceso de resolución de máquinas de diversas plataformas de ciberseguridad. Aquí encontrarás guías detalladas paso a paso para vulnerar y comprometer máquinas virtuales de plataformas como VulnHub, HackTheBox, TryHackMe y otras.

## Estructura del Repositorio

El repositorio está organizado por plataformas de ciberseguridad. Cada plataforma tiene su propio directorio, y dentro de cada uno, encontrarás subdirectorios para cada máquina resuelta.

```
Hacking-Labs/
├── VulnHub/
│   ├── Darkhole2/
│   │   ├── Resolución Darkhole2.md
│   │   └── assets/
│   └── [Proximamente]
│
├── HackTheBox [Proximamente]
│
└── TryHackMe [Proximamente]

```
