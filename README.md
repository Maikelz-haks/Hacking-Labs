# Hacking-Labs

Repositorio que documenta el proceso de resolución de máquinas de diversas plataformas de ciberseguridad. Aquí encontrarás guías detalladas paso a paso para vulnerar y comprometer máquinas virtuales de plataformas como VulnHub, HackTheBox, TryHackMe y otras.

## Estructura del Repositorio

El repositorio está organizado por plataformas de ciberseguridad. Cada plataforma tiene su propio directorio, y dentro de cada uno, encontrarás subdirectorios para cada máquina resuelta.

```
Hacking-Labs/
├── VulnHub/
│   ├── Darkhole2/
│   │   ├── Resolución Darkhole2.md
│   │   └── assets/
│   └── [Proximamente]
│
├── HackTheBox [Proximamente]
│
└── TryHackMe [Proximamente]

```

### Descripción de Directorios

- **Plataforma/**: Directorio principal para cada plataforma (ej. VulnHub, HackTheBox).
- **Máquina/**: Subdirectorio para cada máquina específica.
- **Resolución [Máquina].md**: Archivo Markdown con la guía detallada de resolución.
- **assets/**: Carpeta para capturas de pantalla, scripts o archivos auxiliares utilizados en la resolución.

## Cómo Usar Esta Guía

1. **Navega por las plataformas**: Elige la plataforma de interés y selecciona la máquina que deseas resolver.
2. **Lee la resolución**: Abre el archivo `Resolución [Máquina].md` para seguir los pasos detallados.
3. **Utiliza los assets**: Si hay archivos adicionales en la carpeta `assets/`, descárgalos y úsalos según se indique en la guía.
